MediaRouter
EN 中文 GitHub npm

多媒体生成路由 SDK

用一套稳定接口，接图片、视频、音频和 3D provider。

MediaRouter 统一请求模型、尺寸系统、异步任务、批量拆分和输出结构， 让产品团队不必为每个 provider 反复重写一套接入逻辑。

稳定层与逃生门

共享参数保持稳定，provider 私有参数放到专属出口。

统一请求模型

跨 provider 的公共字段集中放在共享 input 与 options。

插件式 provider

既能直接用内置适配，也能实现你自己的 facade。

一致任务生命周期

轮询、等待、批量拆分、归一化输出都走同一套逻辑。

快速开始

今天先接一个 provider，明天换平台时不用重写业务层。

import { createMediaRouter } from "@miragari/providers"

const client = createMediaRouter({
  provider: "openai",
  providers: {
    openai: process.env.OPENAI_API_KEY!,
  },
})

const result = await client.generateImage({
  prompt: "一张电影感的白色台灯产品图",
  width: 1024,
  height: 1024,
  quality: "high",
})
